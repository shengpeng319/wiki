最近在GitHub上有个很火的概念叫"蒸馏"——把一个大模型的核心知识提取出来，压缩到一个小模型里。

我就想：投资领域最好的"大模型"，不就是那些管着几百上千亿美金的顶级基金经理吗？

于是我花了一些时间，把全球10位最有代表性的投资大师的公开资料——致投资者信、13F持仓报告、访谈、著作——做了一次系统性的"蒸馏"。

结果发现：风格完全不同的10个人，居然有5个几乎一模一样的共识。

# 01

# 先看看这10位是谁

大师 管理规模 一句话风格

-   桥水 (Ray Dalio) $1200亿 不预测，做准备
    
-   巴菲特 $9000亿 好价格买好公司，然后不动
    
-   文艺复兴 (Jim Simons) $800亿 让数据说话，不要有观点
    
-   AQR (Cliff Asness) $1000亿 因子有效，但不是每天有效
    
-   David Tepper $200亿 市场恐慌时，闭眼买
    
-   索罗斯 已退休 市场永远是错的
    
-   Bill Ackman $180亿 5只股票，每只都能写200页报告
    
-   Howard Marks $1900亿 控制风险不是回避风险
    
-   高瓴 (张磊) $600亿 做时间的朋友
    
-   ARK (Cathie Wood) $150亿 颠覆性创新会改变一切
    

这里面有价值派、量化派、宏观派、逆向派、成长派——方法论天差地别。

但蒸馏完以后，我发现底层逻辑惊人地一致。

# 02

# 共识一：有系统，不靠拍脑袋

桥水有全天候系统。 文艺复兴有量化模型。 AQR有因子框架。 巴菲特有选股清单（护城河+ROE+低负债）。 Ackman每只持仓都有200页研究报告。

没有一个人靠"感觉"做投资。

他们都有一套明确的、可复现的决策流程。区别只是有些人写在代码里（文艺复兴），有些人写在脑子里（巴菲特），但本质都是系统。

很多散户的问题恰恰在这里：

买的时候靠"听说这个不错"， 卖的时候靠"感觉不太对"， 复盘的时候靠"下次一定"。

没有系统的投资，本质上是在用运气对抗概率。

# 03

# 共识二：风控比选股重要

这个共识是最出乎我意料的。

你可能觉得这些大佬最厉害的是"选得准"。但蒸馏完发现，他们花在"怎么不亏"上的精力，远超"怎么多赚"。

-   Dalio："分散化是投资的圣杯"——全天候的核心不是选资产，而是让每种资产的风险贡献相等
    
-   Marks："在好的时候表现好不难，在坏的时候不亏太多才是真本事"
    
-   巴菲特：永远保持$1000亿+现金，不是因为看空，是等极端机会
    
-   Tepper：每笔交易都有对冲，从不裸多
    
-   索罗斯："我之所以有钱，是因为我知道什么时候自己错了"
    

而ARK就是最好的反面教材：

2021年涨到天上，2022年跌了80%。-80%需要涨400%才能回本。

为什么？因为她没有风控。

她的理由是"我们是5年投资者"。但5年后基金规模已经从$600亿缩水到$150亿，投资者早就走了。

你可以选错股票，但你不能没有风控。选错了可以换，没有风控可能连换的机会都没有。

# 04

# 共识三：有thesis，能验证，敢认错

Ackman是这一点的极致代表。

他的每只持仓都有：

为什么买（核心论据）

催化剂是什么（什么事件会兑现）

什么条件下退出（论据失效的标准）

2020年COVID刚爆发时，他花$2700万买了一堆信用违约互换（CDS）。他的逻辑很简单："如果我对疫情的判断是对的，这些东西值100倍。"

三周后，他赚了$26亿。

但同样是Ackman，2015年在Valeant上亏了$40亿。亏完以后他说了一句很诚实的话："我不再做我不完全理解的交易。"

关键不是他赚了多少或亏了多少，而是：

他每次都说得清楚自己为什么买、什么情况下会走。

对比很多散户：买的时候说不清理由，跌了以后换一个理由继续拿，最后亏了说"被庄家割了"。

说不清为什么买，就不应该买。说不清什么时候卖，就不知道什么时候会亏到不可收拾。

# 05

# 共识四：周期意识

Howard Marks把这个讲得最清楚。

他有一个"钟摆理论"：市场就像钟摆，永远在过度乐观和过度悲观之间摆动。它很少停在"合理"的中间位置。

2000年：所有人都觉得互联网会改变一切 → 泡沫破裂

2007年：所有人都觉得房价永远涨 → 金融危机

2021年：所有人都觉得创新股会改变一切 → ARK跌80%

每一次都一样，但每一次大家都觉得"这次不一样"

Dalio用他的经济象限模型把周期系统化了——增长×通胀的4种组合，每种组合有利好和利空的资产。

Tepper更直接："政府是最后的后盾。当市场恐慌到政府不得不出手的时候，就是买入的时候。"

周期不是可以精确预测的。但你可以知道"现在大概在什么位置"。

过度乐观时少买，过度悲观时多买。这个道理谁都懂，但能做到的人很少——因为恐慌的时候，你的身体会告诉你"快跑"。

这就是为什么需要系统。系统不会恐慌。

# 06

# 共识五：长期 > 短期

巴菲特："我最喜欢的持有期是永远。" 张磊："长期主义不是长期持有，是长期创造价值。"

但这里要注意一个例外：文艺复兴。

Jim Simons的大奖章基金持有期是分钟到天。年化回报66%，是所有大师里最高的。

这说明什么？

"长期"不是教条。关键是选择适合自己的时间框架，然后在这个框架里保持纪律。

巴菲特的长期是"找到好公司拿着不放"

文艺复兴的长期是"找到统计规律重复执行30年"

两者都是长期主义，只是颗粒度不同

对普通投资者来说，最危险的不是选错时间框架，而是没有时间框架—— 买的时候说长期，跌了两天就想卖； 说做价值投资，涨了20%就想落袋为安。

没有时间框架的投资，不是投资，是赌博。

# 07

# 我能有这些skills么？

你可能会说：这些大佬管着几百亿美金，他们的方法论我怎么用？

确实不能照搬。但底层逻辑是通用的：

建立你自己的系统——哪怕只是一个选股清单+止损规则+复盘习惯

风控第一——不要在一只股票上押全部身家

每笔投资说清楚为什么——写下来，以后验证

知道市场会摆动——贵的时候少买，便宜的时候多买

选一个时间框架并坚持——不要既想做长期又天天看盘

这5条，任何一条单独执行，都能让你的投资水平超过90%的散户。

5条全部执行？那就是大师级别的起点。
