# Vibecoding 工作流

## 概述

Vibecoding = 跟着感觉走，用自然语言驱动 AI 编程的工作方式。由 NS Ropy 团队开发的 Pro Code 工具将此发挥到极致。

## Pro Code 核心数据

- **GitHub 星标**：10.9 万+
- **AI 准则**：28 个
- **技能模块**：125 项
- **命令记忆**：60 个
- **覆盖场景**：需求撰写、代码生成、Bug 检查、测试执行、代码审查

## 关键特征

1. **不是零散的提示词** — 完整的 28 个 agent 协作体系
2. **覆盖全链路开发** — 从写需求到代码审查一条龙
3. **与 Cursor、Cox 深度集成** — 无缝衔接现有工具
4. **号称提效十倍** — 特别适合 Web 开发场景

## 原文摘要

> NS Ropy 团队把 28 个 agent、125 个 skills、60 个命令记忆、优化安全扫描、研究优先工作流全部打包覆盖需求、写代码、查bug做测试审代码这些高频场景，pro code cursor cox 都能接上。
