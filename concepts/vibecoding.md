# Vibecoding

由 NS Ropy 团队开发的 Pro Code 工具提出的编程新范式，获得 10.9 万 GitHub 星标。核心理念：AI 不是辅助工具，而是主导编程过程的核心引擎，开发者更多扮演"决策者"和"审查者"角色。

## 核心特点

- **28 个 AI 准则** — 编程决策的规范框架
- **125 项技能模块** — 覆盖全链路开发场景
- **60 个命令记忆机制** — 持续学习项目上下文
- **完整工作流体系** — 优化、安全扫描、研究优先级管理

## 覆盖场景

- 需求撰写
- 代码生成
- Bug 检查
- 测试执行
- 代码审查

## 技术集成

与 Cursor、Cox 等工具深度集成，实现无缝开发体验，Web 开发据称提效达 10 倍。
